# Notelab Monorepo

## Apps

- `apps/web`: Vite web client and Cloudflare static worker entrypoint
- `apps/desktop`: Tauri desktop shell

## Packages

- `packages/connectors`: shared connector implementations
- `packages/markdown-text-splitter`: shared markdown splitting utilities

## Common Commands

- `npm run dev:web`
- `npm run build:web`
- `npm run dev:desktop`
- `npm run build:desktop`
