Notelab is a workspace for notes, pages, databases, comments, AI workflows, and integrations.

The product has two deployment targets:

- **Notelab Cloud** runs on hosted infrastructure with the Cloudflare adapter.
- **Self-hosted Notelab** runs as a Docker Compose stack with Caddy, Notelab, Postgres, and MinIO.
